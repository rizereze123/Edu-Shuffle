Created by Dhimaz Purnama Adjhi, Edu-Shuffle is a Shuffle/Random For Student Needs program created to address and solve problems that are often common in students. Such as The Division of Group Members, The Order of Group Presentations, and Even the Election of the Chairman or Person in Charge. This program has many advantages such as fair, efficient, fast, and does not drain a lot of time energy. The program is built with several programming language technologies in it such as python, javascript, and css. Always Separate the Names of the candidate participants or groups to be randomized using spaces, and Use just one word. The main point of this program is in a library function called random in python, I merged the python language into html using py-script.
